Cyber Service Tracker 📑
  A Cloud-Based Financial Ledger System designed to track daily transactions, commissions, and capital for a small-scale digital service 
  business (Cyber Cafe).

🚀 Key Features
Real-time Synchronization: Built with Firebase Realtime Database to sync shop data instantly between multiple locations (Hyderabad and Bihar).

Automated Financial Tracking: Automatically calculates and manages Opening Balance, Cash-in-hand, and Digital Assets (Digipay, Airtel, HDFC, etc.).

Commission Management: Includes a dedicated "Commission Tracker" to separate actual profit from total transaction volumes.

Persistent Data Storage: Ensures zero data loss on browser refresh or device changes using cloud persistence.

Mobile-First Responsive UI: Developed with a clean and minimal interface for quick data entry on smartphones during busy shop hours.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom Styling)

Logic: JavaScript (ES6+)

Backend: Google Firebase (Realtime Database)

Deployment: Firebase Hosting

📈 Business Impact
This tool replaces traditional paper-based registers with a digital solution, providing 100% accuracy in daily closing balances and helping in long-term financial analysis of the shop's growth.
